Fonctionnement du thème
Le thème des composants définit grâce à une liste de variables CSS et de classes préparées le style des composants de la librairie.
Pour fonctionner, le fichier CSS du thème doit être importé dans le fichier "index.html".
Vous pouvez utiliser un des thèmes préfaits de la librairie qui se trouve dans le dossier "themes" du paquet ou créer le votre grâce à l'outil présent en-dessous de la page.
Si vous souhaiter modifier le thème pour un composant particulier, vous pouvez changer la valeur d'une des variables CSS au sein du CSS du conteneur du composant désiré.
